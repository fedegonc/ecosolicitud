Demostración de Slots en Layout Base

Slot: layout:fragment="content"

Este es el slot principal donde va el contenido de cada página.

Todo lo que ves aquí está dentro del slot layout:fragment="content".

Este slot se renderiza dentro del <main> en el layout base.

Elementos del Layout Base
  • Header/Navbar: Incluido con th:replace="~{fragments/navbar :: navbar}"
  • Contenido Principal: Insertado con layout:fragment="content"
  • Footer: Incluido con th:replace="~{fragments/footer :: footer}"
  • Script de Navegación: Incluido con th:replace="~{fragments/navbar :: navbarScript}"
Estructura de Slots 
<!-- En el head del layout.html -->
<th:block layout:fragment="styles"></th:block>

<!-- En el body del layout.html -->
<div layout:fragment="content"></div>

<!-- Al final del body del layout.html -->
<th:block layout:fragment="scripts"></th:block>
